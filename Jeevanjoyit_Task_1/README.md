# 🎙️ Voice Assistant

A simple Python-based Voice Assistant that listens to voice commands and performs basic tasks such as telling the time, date, searching Google, opening YouTube, and telling jokes.

---

## 📌 Features

✅ Voice Recognition using SpeechRecognition

✅ Text-to-Speech responses using pyttsx3

✅ Greets the user

✅ Tells the current time

✅ Tells today's date

✅ Searches Google

✅ Opens YouTube

✅ Tells random jokes

✅ Exits on command

---

## 🛠️ Technologies Used

- Python 3
- SpeechRecognition
- pyttsx3
- PyAudio
- PyJokes

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/Voice_Assistant.git
cd Voice_Assistant
```

### 2. Install Required Libraries

```bash
pip install SpeechRecognition
pip install pyttsx3
pip install pyaudio
pip install pyjokes
```

### For Windows Users

If PyAudio installation fails:

```bash
pip install pipwin
pipwin install pyaudio
```

---

## ▶️ Running the Project

Execute the following command:

```bash
python main.py
```

After running, the assistant will start listening for your voice commands.

---

## 🎤 Supported Voice Commands

| Voice Command | Action |
|--------------|---------|
| Hello | Greets the user |
| What time is it | Tells current time |
| What is today's date | Tells current date |
| Search Python programming | Searches Google |
| Open YouTube | Opens YouTube |
| Tell me a joke | Tells a random joke |
| Bye / Exit | Closes the assistant |

---

## 💻 Example Interaction

```text
Assistant: Hello! I am your voice assistant.

User: Hello

Assistant: Hello Jeevan!

User: What time is it?

Assistant: The time is 10:45 AM

User: Tell me a joke

Assistant: Why do programmers prefer dark mode?
```
